Winworldpc

Winworldpc est un site hébergeant des logiciels abandonnés comme :

  • Des vieux systèmes comme :
    • Anciens Winows
    • Bêtas d’anciens Winows
    • Anciens Unix
    • Bêtas d’anciens Unix
    • DOS
    • Bêtas de DOS
    • Anciens MacOS
    • Bêtas d’anciens MacOS
    • OS/2
    • Bêtas d’OS/2
  • D’anciens jeux
  • D’anciens outils de développement
  • D’anciennes applications
  • D’anciens utilitaires

Si vous téléchargez sur ce site, le mirroir 2 est conseillé car plus rapide. Il faudra aussi utiliser 7-zip pour décompresser les fichiers téléchargés.
